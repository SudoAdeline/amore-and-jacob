# Amore & Jacob — Jazz Duo

A high-end, immersive website for the jazz duo **Amore & Jacob**, built with a vintage Hollywood noir aesthetic.

## Design

- **Aesthetic**: Cherry noir — black & white film photography, cherry red bold serif typography, film grain overlays
- **Inspiration**: Bedouin's Daughter editorial layouts, Saint Laurent navigation, Marilyn Monroe-era glamour
- **Typography**: Playfair Display (display/headings), Inter (body)
- **Color Palette**: Near-black `#0a0a0a`, warm off-white `#f5f0eb`, deep cherry `#7A1018`

## Structure

| Section | Description |
|---------|-------------|
| **Navigation** | Saint Laurent-style centered logo with minimal links |
| **Hero** | Full-screen title card (placeholder for duo portrait) |
| **Adeline Amore** | Full-bleed B&W portrait with massive cherry red name overlay, bio on hover |
| **Pierre Jacob** | Full-bleed B&W portrait with massive cherry red name overlay, bio on hover |
| **Performances** | Horizontal scrolling film strip reel with video clips |
| **Inquiry** | Minimal contact form (email + message) |

## Features

- Animated film grain overlay across the entire site
- Heavy grain texture on select portraits for vintage texture
- Parallax scroll effect on artist names
- 1-second hover delay before bio reveal
- Horizontal auto-scrolling video reel (pauses on hover)
- Scroll-triggered fade-in animations
- Fully responsive (mobile tap-to-reveal bios)

## Tech

Single `index.html` file — no build tools, no frameworks, no dependencies. Pure HTML, CSS, and vanilla JavaScript. Google Fonts loaded via CDN.

## Assets

- `Adeline Profil.png` — Adeline Amore portrait (B&W)

## Local Development

Open `index.html` in any browser. No server required.
